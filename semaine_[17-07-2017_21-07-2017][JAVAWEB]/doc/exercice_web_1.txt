exercice web 1A
---------------
ecrire une petite application web avec un formulaire
permettant de donner le resultat d'un calcul (servlet/jsp)
2 champ pour les deux operandes
1 champ combobox pour l'operateur (multiplication, division, addition...)
soumet
 page qui affiche l'operation effectuée et le resultat
 + un lien pour revenir sur le formulaire
 
 gerer une page affichant une erreur en cas de saisie invalide
 par exemple: division par 0
